Inserting symbols which are not shown on the keyboard can be done by choosing
Symbol from the Insert tab or setting the font from a Symbol font such as Wingdings.

 

Inserting a symbol

1.    Click in the document where you want the symbol or character to appear

2.    Click on the Insert tab

3.    Choose More Symbols if the one you need is not showing

4.    Click in the Font list arrow

5.    Choose a new font
e.g. Wingdings, Wingdings 2

6.    Select the required symbol

7.    Click Insert

 

The size of the symbol can be increased
or decreased – but do not change the font
itself.

 

 

 

 

Some useful shortcuts to insert symbols can be seen on the Symbol dialog box.

 

Choose Insert – Symbol – More Symbols – Special Characters tab

Alt+Ctrl+C   for the copyright symbol ©

Alt+Ctrl+R   for the registered symbol ®

Alt+Ctrl+T   for the trademark symbol ™
 
 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


Some useful Wingdings characters are shown below

 

Shift+8     for an envelope symbol *

Shift+9     for a telephone symbol   (

Shift+2     for a pair of scissors        "

Shift+J     for a smiley face     J

Shift+K    for a straight face    K

Shift+L     for a sad face        L
