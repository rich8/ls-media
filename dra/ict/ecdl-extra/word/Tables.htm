Inserting a table

1.   Click where you want the table in your document

2.   Click on the Insert tab

3.   Click on the Table button

4.   Drag out the number of rows and columns you need for the table

5.   The table is inserted on the page

 

 

Gridlines and Borders

When a table is first created it may have borders showing – the heavy lines showing on the table grid which will appear on a printout.

To remove borders

1.    Select the entire table from the cross in the top left-hand corner

2.    Click on the Table Tools tab

3.    Click on Design

4.    Click on the Borders button

5.    Choose No Border

 

 

When the borders have been removed it will leave the gridlines showing

 

To remove gridlines

1.    Select the entire table from the cross in the top left-hand corner

2.    Click on the Table Tools tab

3.    Click on Design

4.    Click on the Borders button

5.    Choose View Gridlines

 

It is helpful to keep the gridlines turned on while working on a table so that you can see the construction of the table – remember gridlines do not appear on a printout.

 

 

You can also turn gridlines on and off from the

Table Tools – Layout tab – View Gridlines button

 

 

 


Selecting rows and columns

Row         Click in the left margin next to the row you want to select

                 Drag down to select multiple rows

Column   Click above the column you want, very near the top line

                 Drag across to select multiple columns

Deleting and Inserting Rows and Columns

1.    Click in the table where you want a new row or column

2.    From the Table Tools tab – choose Layout 

3.    Click Insert Above button to insert a row above the cell where you clicked.

4.    If you click Insert Below, Insert Left or Insert Right, a row or column is inserted into the position you specified.

5.    If you want to delete a row or column, click one of the cells that belong to the row or column you want to delete.

6.    Click Delete button on the ribbon, and then click Delete Rows or Delete Columns to delete the row or column.

 

 

 

 

 


Resizing Rows and Columns

·         To set the row height to a specific measurement, click a cell in the row that you want to resize. On the Layout tab, in the Cell Size group, click in the Table Row Height box, and then specify the height you want.

 

 

 

 

 


·         To change the width to a specific measurement,
click a cell in the column that you want to resize.

On the Layout tab, in the Cell Size group, click in the Table Column Width box, and then specify the options you want.

 

 

Merging Cells

Complex table layouts sometimes need merged or split cells.

1.      Select the cells to be merged or joined together as one cell

2.      Click the Merge Cells button

 

 

 


3.   The cells are merged

4.   To reverse the merge you need to use the Split Cells button

 

Changing Borders  style and colours

·          When you want to change the design of a table border from the Design tab use the Borders and Shading option at the bottom of the Borders button list.

·          Choose the Custom option first, then select the style, weight and colour of the line before applying the design to the table grid in the preview section, by clicking on the outer border or inner gridlines.

Table alignment

Alignment of text entries in a table follow the usual rules – select the text and use the alignment buttons on the Home tab. To align an entire row or column select it first.

 

To align the entire table itself in the page – select with the cross in the top left-hand corner of the table, then use the alignment buttons on the Home tab.
